This page provides API prototype to the NFDI4Objects Knowledge Graph. The Knowledge Graph integrates research data about cultural heritage objects. The graph consists of two databases with individual APIs each:

  • an RDF triple store with SPARQL query API at /api/sparql

  • a Labeled Property Graph with Cypher Property Graph API at /api/cypher

In addition, data about collections of research data in the graph can be retrieved as Linked Open Data (LOD) for collection URIs in the https://graph.nfdi4objects.net/collection/ (n4oc) namespace.

This is a first version, being worked on! Triple store and labeled property graph may contain different subsets of the knowledge graph and data model is not stable yet. Please consult the manual for details!
