This is a first version, being worked on!

This page provides API prototype to the NFDI4Objects Knowledge Graph. The Knowledge Graph integrates research data about cultural heritage objects. The following APIs are provided:

  • SPARQL query API to the RDF triple store at /api/sparql

  • Linked Open Data (LOD) about collections of research data in the graph for collection URIs in the https://graph.nfdi4objects.net/collection/ (n4oc) namespace.

Please consult the manual for background and details!
