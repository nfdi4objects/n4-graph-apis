This page provides an API prototype to the NFDI4Objects Knowledge Graph. The Knowledge Graph integrates research data about cultural heritage objects within NFD4Objects. The following APIs are provided:

  • SPARQL query API to the RDF triple store at /api/sparql (see SPARQLExplorer for inspection).

  • Linked Open Data (LOD) about collections of research data in the graph for collection URIs in the https://graph.nfdi4objects.net/collection/ (n4oc) namespace and terminology URIs in the http://bartoc.org/en/node/ namespace.

    • {% if config.cypher %}

  • experimental Cypher Property Graph API to a Labeled Property Graph at /api/cypher

    • {% endif %}

Please consult the manual for background and details!
