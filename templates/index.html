This page provides API prototype to the NFDI4Objects Knowledge Graph. The Knowledge Graph consists of two databases with individual APIs each:

  • an RDF triple store with SPARQL query API at /api/sparql

  • a labeled property graph with Cypher Property Graph API at /api/cypher

In addition data can be retrieved

  • as Linked Open Data (LOD) Linked Open Data (LOD) for collection URIs in the https://graph.nfdi4objects.net/collection/ (n4oc) namespace

Triple store and labeled property graph may contain different subsets of the knowledge graph. Please consult the manual for details!
