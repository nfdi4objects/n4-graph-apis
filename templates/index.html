This page provides API prototype to the NFDI4Objects Knowledge Graph. The Knowledge Graph consists of two databases with individual API each:

  • an RDF triple store with SPARQL query API at /api/sparql.
    Try online!
  • a labeled property graph with Cypher Property Graph API at /api/cypher.
    Try online!

Triple store and labeled property graph contain different subsets of the knowledge graph. Please consult the manual for details!
